# 2048 Game Clone

![2048-game-img](image-3.png)

A game created by following a [tutorial](https://youtu.be/4NFZwPhqeRs?si=BCOmYUt5u1YFP46O) to learn the basics of Unity.

## How to Play

`W` or `↑` to move up

`A` or `←` to move left

`S` or `↓` to move down

`D` or `→` to move right

`New Game` to restart the game
